Les bibliothèques nécessaires au bon fonctionnement du programme sont les suivantes:
  PyQt5 V5.14.0
  SQLite3 (inclus dans les versions standards de Python au dessus de la V2.5), sinon la bibliothèque est disponible ici : https://www.sqlite.org/download.html
  
  

Pour lancer une partie du jeu, il faut executer le module Jeux.py.

Pour vérifier les tests, il faut executer le programme TestJoueur.py.
